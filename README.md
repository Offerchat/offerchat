Offerchat Gem
=============

Add live chat to your website for free with [Offerchat](https://www.offerchat.com).

Sign up for a free account [here](https://new.offerchat.com/signup_wizard)

Usage
-----

* Add `gem 'offerchat'` to your `Gemfile`
* bundle install
